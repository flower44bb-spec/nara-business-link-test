# 管理者向け利用状況ダッシュボード

管理者ページに、無料枠から有料枠への切替判断に使う利用状況を表示します。

## 表示項目

- 累計閲覧数
- 直近30日の閲覧数
- 直近30日のユニーク訪問者数
- 直近30日のログイン利用者数
- Supabase認証の月間利用者数
- 承認済み会員数
- 登録会員総数と承認待ち会員数
- 直近30日の新規会員数
- 登録事業者総数、承認済み・承認待ち事業者数
- 全投稿数
- いいね数
- DM送信数
- DB概算容量
- 画像ストレージ概算容量とファイル数
- 日別アクセス数推移
- よく見られているページ
- 登録転換率の参考値
- 会員利用率の参考値

IPアドレスは保存しません。ブラウザ内で生成した匿名ID、閲覧ページ、日時、端末種別を記録します。
閲覧記録は専用のDB関数だけを通して追加し、閲覧者へ履歴の読み取り権限は付与しません。

## Supabase側の作業

SQL Editorで次のmigrationを実行してください。

```text
supabase/migrations/202606150001_admin_analytics.sql
```

アクセス数はmigration実行後から記録されます。過去分は遡って集計できません。

## 管理画面だけでは取得できない項目

以下は各サービスのUsage画面で確認します。

- SupabaseのDB容量
- Supabase Storageの画像容量
- Supabaseの転送量と月間利用者数
- Vercelのデータ転送量
- Vercelのリクエスト数
- Vercel Functionsの使用量

これらの転送量・実行量をサイト内へ自動表示するには、各サービスの管理APIトークンと安定したUsage APIが必要です。
認証情報をブラウザへ置くことはできないため、設定する場合はVercelのサーバー環境変数で管理します。

無料枠上限の70〜80%を継続して超える場合は、有料化を検討する目安です。
また、Vercel Hobbyは個人・非商用向けのため、公式業務として運用する場合は使用量にかかわらずプラン条件を確認してください。
